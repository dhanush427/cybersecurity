Ten Ways to Prevent Cyber Crimes


  1. Train employees in cyber security principles.
  2. Install, use and regularly update antivirus and antispyware software on every computer used in your business.
  3. Use a firewall for your Internet connection.
  4. Download and install software updates for your operating systems and applications as they become available.
  5. Make backup copies of important business data and information.
  6. Control physical access to your computers and network components.
  7. Secure your Wi-Fi networks. If you have a Wi-Fi network for your workplace make sure it is secure and hidden.
  8. Require individual user accounts for each employee.
  9. Limit employee access to data and information and limit authority to install software.
  10. Regularly change passwords.

How do we prevent it?


  • Ransomware - An attack that locks the user's computer system files. The attacker than demands money from the user in order to unlock the computer.
  • Malware - A file or program used to damage the user's computer.
  • Social Engineering - An attack that relies on users to trick them into breaking security procedures in order to gain important information.
  • Trojans - A computer program that misleads users from its actual purpose.
  • Cyber Crime - A crime that involves a computer and a network.
  • Vulnerability - A weakness that can perform an unauthorized action to a computer system.
  • Eavesdropping - That act of secretly listening to a private conversation without the user's permission.
  • Viruses - A type of software that replicates itself by modifying other computer programs.
  • Worms - A computer program that replicates itself so it could spread through other computers.
